## 核心功能

### 景點探索與搜尋
![景點探索與搜尋](/public/search.gif)

- **即時瀏覽與篩選**：支援關鍵字搜尋，使用者可快速尋找台北熱門景點。
- **效能優化**：針對景點圖片進行預處理與非同步載入，提升頁面首渲染速度。

### 行程預約系統
![行程預約系統](/public/booking.gif)

- **動態行程規劃**：使用者可選擇特定景點、日期與時間（上半天/下半天），系統自動計算預約費用。
- **預約清單管理**：提供即時的預約資訊呈現，方便使用者在下單前確認行程細節。

### 會員管理與安全支付
![登入畫面](/public/login.png)
![安全支付](/public/order.gif)

- **會員身份驗證**：整合註冊與登入功能，使用 JWT (JSON Web Token) 機制確保個人資料的安全傳輸。
- **TapPay 金流串接**：整合 TapPay 代碼化支付技術，提供安全、便捷的信用卡線上下單流程。

## 快速開始

1. **環境設定**：安裝 Python 3.11+ 並安裝依賴項。
   ```bash
   pip install -r requirements.txt
   ```

2. **資料庫配置**：在 MySQL 中建立資料庫，並執行建表指令。

3. **啟動專案**：
   ```bash
   uvicorn app:app --reload
   ```
